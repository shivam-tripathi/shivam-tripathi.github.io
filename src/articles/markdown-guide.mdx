## Text Formatting

### Bold and Italic

**This is bold**
*This is italic*
***This is bold and italic***

### Strikethrough

~~This is strikethrough~~

## Headings

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

## Lists

### Unordered Lists

- Item 1
- Item 2
  - Nested item 1
  - Nested item 2
- Item 3

### Ordered Lists

1. First item
2. Second item
3. Third item

## Code

### Inline Code

Use `const x = 5;` for inline code.

### Code Blocks

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

## Links and Images

[Visit Google](https://google.com)

## Blockquotes

> This is a blockquote
>
> It can span multiple lines
