# Example Blog Post in MDX

<p style="width: 100%; text-align: right; padding-right: 20px"><em>10 November 2025</em></p>

This is an example of how to write blog posts as MDX files directly in your project!

## Benefits of Using MDX

- **Easier to maintain**: All your blog posts are in one place
- **No build step needed**: Just write and refresh
- **Better formatting**: Use proper markdown editors
- **Version control**: Track posts in git naturally

## How to Use

1. Create a new `.mdx` file in `src/articles/`
2. Write your content in markdown
3. Import it in `src/data/blogs.js` using the `importMDXArticle()` utility
