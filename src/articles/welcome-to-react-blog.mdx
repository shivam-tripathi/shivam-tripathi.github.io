## What You Can Do

### Write in Markdown

Write your blog posts in clean, simple Markdown syntax:

```markdown
# Heading
**Bold** and *italic* text
```

### Include Code Blocks

With syntax highlighting:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

### Use Math

Write mathematical equations using LaTeX:

$$E = mc^2$$

## Getting Started

1. Create a new `.mdx` file in the `src/articles/` directory
2. Add your blog post to the `blogPosts` array in `blogs.js`
3. Run `npm run build` and deploy!
